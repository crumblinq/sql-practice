# Средняя стоимость фильмов с несколькими условиями

- [Схема базы данных](../database-scheme-ru.png)
- [Подробное описание таблиц](../database-description-ru.pdf)

## Задача

Нужно вычислить среднюю стоимость фильмов, отобранных по двум условиям:
- их длительность не превышает среднюю длительность фильмов в базе,
- в них снималось более 12 актёров и актрис.

## Решение

Прибегнем к декомпозиции и разобьем задачу на подзадачи:
1. Найдём среднюю длительность всех фильмов в базе.
2. Найдём фильмы, в которых снималось более 12 актёров и актрис.
3. Отфильтруем таблицу по средней длительности фильмов.
4. Вычислим среднюю стоимость отобранных фильмов.

Шаг 1: Найдём среднюю длительность всех фильмов в базе

```sql
SELECT AVG(length) 
FROM movie AS m;
```

Промежуточный результат 1:

| avg |
|---|
| 115.272|

Шаг 2:  Найдём фильмы, в которых снималось более 12 актёров и актрис

```sql
SELECT m.film_id
FROM movie AS m 
LEFT JOIN film_actor AS fa ON m.film_id = fa.film_id
GROUP BY m.film_id
HAVING COUNT(actor_id) > 12;
```

Промежуточный результат 2:

| film_id |
|--------:|
| 508 |
| 714 |
| 249 |
| 188 |
| 606 |
| 87 |
| 146 |

Шаг 3: Осталось вычислить среднюю стоимость фильмов, пока без добавления условий

```sql
SELECT AVG(rental_rate)
FROM movie AS m;
```

Промежуточный результат 3:

| avg |
|---|
| 2.98|

Шаг 4: Теперь можно указать WHERE и добавить два подзапроса

```sql
SELECT AVG(rental_rate)
FROM movie AS m
WHERE film_id IN (SELECT m.film_id
                  FROM movie AS m 
                  LEFT JOIN film_actor AS fa ON m.film_id = fa.film_id
                  GROUP BY m.film_id
                  HAVING COUNT(actor_id) > 12)
 AND length <= (SELECT AVG(length) 
                FROM movie AS m);
```

## Результат

| avg |
|---|
| 2.32333|
